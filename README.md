# LocalGems

Фронтенд для сервиса поиска локальных достопримечательностей.

## Технологии

- React 19 + TypeScript
- Vite
- Tailwind CSS 
- Zustand 
- Яндекс.Карты
- Axios

## Основные возможности

- Интерактивная карта с цветными маркерами
- Фильтрация мест по категориям и поиску
- Избранное (сохраняем места)
- Маршруты (добавление, удаление, экспорт в Яндекс.Карты)
- Авторизация (вход / регистрация)
- Личный кабинет модератора (управление местами)
